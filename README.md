# PKM Box Store

한국어판 포켓몬 카드 박스 판매를 위한 커머스 프로젝트입니다.

## Tech Stack

- Backend: Spring Boot, Java 17, JPA, MySQL
- Frontend: Next.js, TypeScript
- Payment: Toss Payments
- Image Storage: AWS S3
- Deploy: AWS

## Core Features

- 상품 목록 / 상세 조회
- 관리자 상품 등록 / 수정 / 삭제
- S3 상품 이미지 업로드
- 장바구니
- 주문 생성
- 재고 관리
- Toss Payments 결제
- 관리자 주문 관리
