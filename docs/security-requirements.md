# Security Requirements

Status values:

- `Implemented`: 코드 근거가 확인됨
- `Partially Implemented`: 일부 통제는 있으나 테스트/운영 확인 또는 범위 보강 필요
- `Not Implemented`: 현재 근거 없음
- `Needs Verification`: 코드 또는 운영 환경에서 추가 확인 필요
- `Deferred`: 현재 범위 밖
